# COVID-19 Data Analysis

Questo repository contiene il codice per l'analisi dei dati COVID-19 utilizzando Python e JupyterLab. L'analisi include l'evoluzione dei casi totali, nuovi casi giornalieri e pazienti ospedalizzati in diverse nazioni europee.

## Descrizione

Il progetto analizza i dati relativi ai casi COVID-19 in Italia, Germania, Francia e Spagna per l'anno 2023. Include script per visualizzare:
- L'evoluzione dei casi totali rispetto alla data.
- Il numero di nuovi casi giornalieri rispetto alla data.
- L'andamento della somma cumulativa dei nuovi casi.
- Confronti tra nazioni riguardo il numero di pazienti in terapia intensiva e ospedalizzati.

## Prerequisiti

- Python 3.x
- JupyterLab
- Librerie Python: pandas, matplotlib
